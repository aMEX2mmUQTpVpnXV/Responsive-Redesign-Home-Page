Redesigning a Puppy Shop Website
Overview
The goal of this webpage is to detail the steps one would go through to redesign a web page.
Choosing a Website
In order to begin redesigning a website we must first find a website to redesign! Websites that have issues regarding usability and accessibility are usually the best candidates.

The website we will be redesigning for this demo is Lancaster Puppies Shop.

The redesigned website can be found by clicking the following: Redesigned Puppy Shop.
Usability Issues
Some issues this website faces regarding usability are:

  • The website does not have a minimalistâ€™s design. The interface is quite cluttered and it is difficult to immediately tell what is happening on the page.
  • There are symbols in the images that are unexplained / unclear.
  • Boxes with rounded edges seem to indicate a web element that is clickable but this is not always true.
  • Inconsistent image sizing leads to text that is not lined up in a uniform manner. Makes the website harder to parse.
  • On mobile it is not clear that the icons are buttons and since there is no text, their function is completely unclear.
  • In the sort by menu it is unclear as to what metric is being used currently.
  • Gratuitous text ex: Puppies for sale in Lebanon , PA
  • When you use price filter it reloads and resizes, preventing you from loosening the restriction.
Accessibility Issues
The website has a lot of contrast issues. A lot of the text is in a light blue color and it does not contrast with the primarily white background enough to make the text easy to read. Other parts of the website have the same issue but with light gray text. The website also has some empty links and links that lead to non functioning or nonexistent webpages.
